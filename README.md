# Food_Delivery_System

Food Delivery System | Java, JSP, Servlets, MySQL
A full-stack web application built using Java, JSP, Servlets, DAO Design Pattern, and MySQL, demonstrating a complete workflow from user login to order placement.

# Project Overview

-> This project simulates a real-time food delivery application where users can:
-> Register and log in.
-> Explore restaurants and menu items.
-> Add items to the cart.
-> Place orders.
-> View order details.
-> The system is designed using the MVC architecture and follows a clean, modular DAO structure for database operations.

# Technologies Used

-> Frontend: HTML, CSS, JSP
-> Backend: Java Servlets
-> Architecture: DAO Design Pattern
-> Database: MySQL
-> Server: Apache Tomcat
-> Tools: Eclipse IDE/ MySQL Workbench

# Key Features

-> User Authentication (Login & Registration)
-> Restaurant Page with Menu Display
-> Add-to-Cart Flow
-> Cart Management
-> Secure Order Placement
-> Modular DAO Layer for DB operations
