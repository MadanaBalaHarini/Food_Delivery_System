# Food_Delivery_System

Food Delivery System | Java, JSP, Servlets, MySQL
A full-stack web application built using Java, JSP, Servlets, DAO Design Pattern, and MySQL, demonstrating a complete workflow from user login to order placement.

# Project Overview

-> This project simulates a real-time food delivery application where users can:
-> Register and log in.
-> Explore restaurants and menu items.
-> Add items to the cart.
-> Place orders.
-> View order details.
-> The system is designed using the MVC architecture and follows a clean, modular DAO structure for database operations.

# Technologies Used

-> Frontend: HTML, CSS, JSP
-> Backend: Java Servlets
-> Architecture: DAO Design Pattern
-> Database: MySQL
-> Server: Apache Tomcat
-> Tools: Eclipse IDE/ MySQL Workbench

# Project Architecture

The application follows the MVC architecture:

-> Model: Java classes representing entities like User, Restaurant, Menu, Order
-> DAO Layer: Handles all database operations using JDBC
-> Controller: Servlets manage business logic and request handling
-> View: JSP pages used for UI rendering

# Project Flow

1. User registers or logs into the system
2. User views available restaurants
3. Selects a restaurant and views menu items
4. Adds food items to the cart
5. Reviews cart and places the order
6. Order details are stored in the database

# Database Details

Database: MySQL

Tables used:
  -> users
  -> restaurants
  -> menu
  -> cart
  -> orders
  -> order_items
